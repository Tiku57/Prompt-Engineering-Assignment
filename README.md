## ğŸ¤– Prompt Engineering Strategy
Instead of a single "do it all" prompt, I utilized a **Modular Meta-Prompt Strategy**:

1.  **The Meta-Prompt:** Established the persona (Senior Front-End Dev), the design system (Slate-900 & Blue-600), and technical constraints (Inline SVGs, Mobile-First).
2.  **Chained Prompts:** Each section was generated individually using the context established in the meta-prompt to ensure visual consistency.

**Key Constraints Applied:**
* *Restricted external icon libraries* to prevent broken links.
* *Enforced specific Unsplash URL parameters* (`fit=crop`) to maintain aspect ratios.
* *Mandated `w-full` and `grid-cols-1` defaults* to guarantee mobile responsiveness.
